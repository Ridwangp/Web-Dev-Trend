Trend WebDev 2018

Intro

Banyak Teknologi baru seperti Web Component atau PWA ( Progressive Web App) yang sangat berguna bagi
Browser-browser di Tahun 2017.
Artikel kali ini akan membahas mengenai Trend Perkembangan Web di Tahun 2018.

1. Mengerti Dasar


Untuk menjadi Web Developer di tahun 2018 dibutuhkan perhatian khusus pada Fundamental. Yaitu masih dengan
Javacscript,HTML dan CSS. Seorang Web Developer perlu mengetahui bagaimana cara Javascript, Html dan CSS berkerja bersama.

2. Node.Js


Masih sama seperti di tahun 2017 Node.Js masih akan populer di tahun 2017. Mengapa harus Node.js?? karena
Node.Js bisa dikatakan adalah bahasa yang digunakan untuk Backend-Developer dan bahasa dasar yang digunakan adalah Javascript, sehingga memungkinkan untuk menguasai Back-End dan Front-End sekaligus.

3. Lebih Mengenal React, Angular and/ or Vue.js


React, Angular dan Vue.Js adalah framework yang digunakan untuk client-side atau yang juga biasa disebut front-end dengan menggunakan basis bahasa javasript, tetapi react tidak bisa dikatakan sebagai framework, lebih tepatnya adalah library.Ketiga hal di atas menciptakan pengalaman pengguna/user experience yang sangat menarik untuk aplikasi web kamu karena semuanya dikontrol dengan JavaScript. Pengguna mungkin terus menyukai aplikasi web yang menarik, sehingga kerangka kerja frontend seperti Angular, React atau Vue.js akan tetap penting di tahun 2018. Mempelajari salah satu atau semuanya akan menjadi ide yang sangat bagus. Mereka memungkinkan dirimu untuk belajar tentang web developmnet dan menciptakan web yang bagus dengan interface yang interaktif

4. Komponen Web & Kompiler Komponen

Pada akhir 2017, Support Browser masih menjadi masalah yang cukup besar, Tetapi pada awal 2018, Masalah ini berkurang. Empat API Web yang membentuk komponen web yaitu, Unsur Khusus, DOM Shadow, Template HTML dan Impor HTML sudah bisa digunakan oleh Browser-browser ternama yang paling banyak digubakan.

Ide pada web components itu sederhana:Untuk apa menggunakan library seperti React untuk membangun komponen yang hanya berfungsi di library itu sendiri, sedangkan dengan menggunakan vanilla JavaScript dapat membangun komponene web yang juga dapat digunakan luar framework.

5. Penggunaan Static Page Generator


Trend berikutnya di tahun 2018 adalah Penggunaaan Static Page Generator, Static Page Generator adalah aplikasi statis yang memungkinkan kita untuk membuat halaman web pada server lokal atau pun built-in server yang berisikan Html,Css dan Javasript. Keuntungan dari Static Page generator adalah, karena sudah multi platform dan juga dapat menggunakan markdown. Contoh beberapa Generator Static Page adalah Jekyll, Hugo, Nuxt.js dan lain-lain

6. Progressive Web Apps

Progressive Web Apps (PWA) merupakan istilah yang mencakup beberapa teknologi yang memungkinkan untuk membuat aplikasi web yang berperilaku sangat mirip dengan aplikasi mobile.

keuntungannya adalah karena mencakup fungsi offline (misalnya, bagian dari aplikasi web yang dapat berfungsi tanpa koneksi Asalkan telah dimuat satu kali) penggunaan perangkat asli fitur (seperti mengakses kamera perangkat).

Kekurangannya adalah kamu harus mempelajari dua bahasa (Java / Android dan Swift / ObjectiveC) dan jangkauannya punan terbatas.

7. Membuat Serverless Apps

Serverless bukan berarti tidak menggunakan server sama sekali, namun menggunakan server hanya jika diperlukan saja, sehingga tidak perlu khawatir mengelola dan memperbarui server. Space pada server pun akan menjadi lebih banyak. Biaya yang dikeluarkan pun lebih sedikit karena tidak perlu membayar saat dalam keaadan idle.

8. Artificial Intelligence

AI adalah bidang yang luas Kamu bisa menyelam ke Python, Machine Learning, Big Data, Deep Learning dan membangun alat AI sendiri. Jika Kamu menguasai semua itu, Kamu akan memiliki masa emas di depan Kamu.
Namun, untuk penggunaan lainnya, kami dapat menyelami lebih banyak alat yang memungkinkan kami menerapkan layanan AI ke semua jenis aplikasi dan model bisnis tanpa persyaratan menjadi ahli AI.
Pertimbangkan alat seperti AWS Lex, layanan yang memungkinkan Kamu membangun bots obrolan cerdas yang dapat Kamu terapkan ke halaman Kamu. Alat seperti ini membuat AI dapat diakses oleh kita semua dan memungkinkan Kamu untuk berfokus pada ide dan logika bisnis Kamu. Dan kita akan melihat banyak alat seperti ini muncul (dan menjadi lebih baik) di tahun 2018 - oleh karena itu ini adalah tren yang mungkin tidak ingin Kamu lewatkan.
